# SemEval Task 4: Narrative Similarity Track A  
### CS 14B — Exploratory Data Analysis

This project contains the full exploratory data analysis (EDA) for SemEval Task 4: Narrative Similarity, Track A. The goal is to analyze a dataset of anchor-A-B text triples, visualize patterns, and prepare for modeling with deep text features. All code and plots are reproducible.

## Setup

1. Place your dataset `dev_track_a.jsonl` in the project directory.
2. Ensure there is a `plots/` folder (or create one) for output PDFs.
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```

---

## Files

- `eda.py` — All EDA and plotting code. Generates and saves all plots.
- `dev_track_a.jsonl` — Dataset file (not included here).
- `plots/` — Directory where all plots are saved as PDF/PNG.
- `README.md` — This file.
- `requirements.txt` — Python dependencies list.
